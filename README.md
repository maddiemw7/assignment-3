# Favorites App

This project is the completed version of the Favorites app for Module 3 of the Mobile Application Development course.

The project includes:

* Model classes for cities, hobbies, and books
* Sample data used throughout the module
* Image assets for the city cards
* Completed screen structure
* Reusable widgets
* Provider-based shared state management
* Local data persistence using SharedPreferences
* Favorites management
* Settings screen
* Theme switching

Before starting, run:

```bash
flutter pub get
```

to download any required dependencies.

Once the project is ready, open main.dart and run the application on a simulator, emulator, or physical device.
