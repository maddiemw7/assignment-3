# Favorites App Skeleton Project

This starter project is provided for Module 3 of the Mobile Application Development course.

The project includes:

* Model classes for cities, hobbies, and books
* Sample data used throughout the module
* Image assets for the city cards
* Basic project structure with screens, widgets, and providers folders

The goal of this starter project is to reduce setup time so we can focus on building the application.

Throughout the module, you will implement:

* Bottom navigation
* Reusable widgets
* Shared state management using Provider
* Local data persistence using SharedPreferences
* Search functionality
* Favorites management
* Theme switching
* Additional category support and code refactoring

Before starting, run:

```bash
flutter pub get
```

to download any required dependencies.

Once the project is ready, open `main.dart` and follow along with the module videos to build the Favorites application step by step.
